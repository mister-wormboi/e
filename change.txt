Обоснование выбора программного обеспечения

1. Ubuntu
Выбрана, потому что это бесплатная и стабильная операционная система, которая подходит для разработки и администрирования.

2. OpenSSH Server
Используется для удалённого подключения к компьютеру и выполнения настройки по сети.

3. LibreOffice
Выбран для создания и редактирования документов, необходимых при выполнении задания.

4. GIMP
Используется для работы с изображениями и является бесплатным аналогом Photoshop.

5. 7-Zip
Нужен для создания и распаковки архивов различных форматов.

6. HardInfo
Позволяет посмотреть информацию о компьютере и установленном оборудовании.

7. ClamAV
Бесплатный антивирус для проверки системы на наличие вредоносных программ.

8. ClamTK
Графическая оболочка для ClamAV, делает работу с антивирусом более удобной.

9. CUPS-PDF
Создаёт виртуальный PDF-принтер, который позволяет сохранять документы в формате PDF.
