1. Обновление списка пакетов
Команда: sudo apt update
Обоснование: нужна, чтобы система получила актуальный список программ перед установкой.

2. Обновление установленных пакетов
Команда: sudo apt upgrade -y
Обоснование: обновляет уже установленные программы до последних версий.

3. Установка необходимого программного обеспечения
Команда: sudo apt install -y openssh-server libreoffice gimp 7zip hardinfo clamav clamtk printer-driver-cups-pdf
Обоснование: устанавливает программы, которые нужны для выполнения задания.

4. Включение службы SSH
Команда: sudo systemctl enable ssh
Обоснование: чтобы служба SSH запускалась автоматически при включении компьютера.

5. Запуск службы SSH
Команда: sudo systemctl start ssh
Обоснование: запускает SSH сразу, чтобы можно было подключаться удалённо.

6. Включение брандмауэра UFW
Команда: sudo ufw enable
Обоснование: включает защиту системы от нежелательных подключений.

7. Разрешение подключения по SSH
Команда: sudo ufw allow OpenSSH
Обоснование: разрешает подключение по SSH через брандмауэр.

8. Создание группы пользователей
Команда: sudo addgroup admins
Обоснование: создаёт группу для удобного управления правами пользователей.

9. Создание пользователя
Команда: sudo useradd -m student
Обоснование: создаёт нового пользователя с домашней папкой.

10. Назначение пароля пользователю
Команда: sudo passwd student
Обоснование: устанавливает пароль для входа под новым пользователем.

11. Создание папки для резервной копии
Команда: sudo mkdir -p /backup
Обоснование: создаёт папку, в которой будет храниться резервная копия.

12. Создание резервной копии каталога /etc
Команда: sudo tar -czf /backup/system_backup.tar.gz /etc
Обоснование: создаёт архив с системными настройками.

13. Проверка состояния службы SSH
Команда: sudo systemctl status ssh
Обоснование: позволяет убедиться, что служба работает.

14. Проверка IP-адреса компьютера
Команда: ip a
Обоснование: показывает сетевые интерфейсы и IP-адрес устройства.

15. Проверка подключения к Интернету
Команда: ping -c 4 8.8.8.8
Обоснование: проверяет, есть ли доступ в интернет.
