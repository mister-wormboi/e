РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

Наименование программы
GIMP

Назначение программы
GIMP — это бесплатный графический редактор, предназначенный для создания, просмотра и редактирования изображений. Программа позволяет работать с фотографиями, рисунками, логотипами и другими графическими файлами.

Запуск программы
1. Откройте меню «Приложения».
2. Перейдите в раздел «Графика».
3. Выберите программу GIMP.
4. После запуска откроется главное окно программы.

Основные функции
- создание новых изображений;
- открытие и редактирование существующих файлов;
- изменение размера изображения;
- обрезка изображений;
- рисование с помощью кистей и других инструментов;
- добавление текста;
- применение фильтров и эффектов;
- экспорт изображений в различные форматы (PNG, JPG, BMP, TIFF и другие).

Порядок работы
1. Выберите пункт «Файл» → «Создать» или «Файл» → «Открыть».
2. Выполните необходимые изменения изображения с помощью инструментов программы.
3. При необходимости сохраните проект в формате XCF.
4. Для получения готового изображения выберите «Файл» → «Экспортировать как», укажите формат и место сохранения.
5. Нажмите кнопку «Экспорт».

Завершение работы
После окончания работы сохраните изменения, если это необходимо, затем выберите «Файл» → «Выход» или закройте окно программы.

Системные требования
Для работы программы требуется установленная операционная система Linux (Ubuntu), не менее 4 ГБ оперативной памяти и около 500 МБ свободного места на диске.

РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

Наименование программы
LibreOffice

Назначение программы
LibreOffice — бесплатный офисный пакет, предназначенный для создания и редактирования текстовых документов, электронных таблиц, презентаций и других файлов.

Запуск программы
1. Откройте меню «Приложения».
2. Выберите раздел «Офис».
3. Запустите LibreOffice или нужный модуль (Writer, Calc, Impress).

Основные функции
- создание текстовых документов;
- работа с электронными таблицами;
- создание презентаций;
- просмотр и редактирование документов;
- экспорт файлов в PDF.

Порядок работы
1. Запустите программу.
2. Создайте новый документ или откройте существующий.
3. Выполните необходимые изменения.
4. Сохраните документ через меню «Файл» → «Сохранить».

Завершение работы
Сохраните изменения и выберите «Файл» → «Выход».

Системные требования
Для работы требуется операционная система Linux (Ubuntu), не менее 2 ГБ оперативной памяти и около 2 ГБ свободного места на диске.

РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

Наименование программы
Visual Studio Code (VS Code)

Назначение программы
Visual Studio Code — бесплатный редактор исходного кода, предназначенный для разработки программ на различных языках программирования. Поддерживает установку расширений для работы с разными технологиями.

Запуск программы
1. Откройте меню «Приложения».
2. Выберите Visual Studio Code.
3. Дождитесь загрузки главного окна программы.

Основные функции
- создание и редактирование исходного кода;
- работа с проектами и папками;
- подсветка синтаксиса;
- установка расширений;
- встроенный терминал;
- работа с системой контроля версий Git;
- отладка программ.

Порядок работы
1. Запустите программу.
2. Выберите «File» → «Open Folder» и откройте папку проекта.
3. Создайте новый файл или откройте существующий.
4. Напишите или измените код.
5. При необходимости используйте встроенный терминал для запуска программы.
6. Сохраните изменения через «File» → «Save».

Завершение работы
Сохраните все изменения, затем выберите «File» → «Exit» или закройте окно программы.

Системные требования
Для работы требуется операционная система Linux (Ubuntu), не менее 4 ГБ оперативной памяти и около 500 МБ свободного места на диске.
